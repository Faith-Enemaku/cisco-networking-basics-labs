# Lab 01 – Wireless Router & Client Configuration

## Objective
Configure a small home network by connecting wired and wireless devices, configuring a home wireless router, enabling DHCP, securing a WLAN, and verifying end-to-end internet connectivity.

## Key Steps
1. **Connect Devices**
   - Connect coaxial cables from the cable provider to the modem and TV.
   - Connect PCs to the home wireless router using Ethernet cables.
   - Connect the cable modem to the router’s Internet port.

2. **Configure Wireless Router**
   - Access the router GUI via Office PC web browser.
   - Enable DHCP for client IP addressing.
   - Change default admin credentials.
   - Set up wireless network (SSID: `MyHome`) and security (WPA2, passphrase: `MyPassPhrase1!`).

3. **Configure IP Addressing and Test Connectivity**
   - Connect Laptop to the wireless network using the configured SSID and passphrase.
   - Verify connectivity by accessing `skillsforall.srv`.
   - Confirm that Office PC and Bedroom PC have proper IP addresses and internet access.
